# Current Project

## [AWS SOC Lab](https://github.com/Vasanth-Cybersec/AWS-SOC-Lab.git) (Work in Progress)

I am currently building a Security Operations Center (SOC) lab environment in AWS to simulate attacker activity and practice threat detection.

### Lab Architecture

* **Windows Server** – Target system generating logs
* **Ubuntu Server** – Attacker / testing machine
* **SIEM (Planned)** – centralized log monitoring
* **Cloud Infrastructure** – AWS EC2

### Goals of the Lab

* Simulate attacker techniques such as:

  * Network scanning
  * Remote login attempts
  * Privilege escalation
  * Service creation
* Collect logs from multiple systems
* Detect malicious behavior using SIEM queries
* Investigate Windows Event Logs and security alerts

### Tools Being Used

* AWS EC2
* Windows Server
* Ubuntu Linux
* Splunk (planned)
* Sysmon
* Nmap
* Metasploit

### Skills Practiced

* Log analysis
* Threat detection
* Incident investigation
* SIEM configuration
* Cloud security monitoring
